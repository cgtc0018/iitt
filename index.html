1日目 - 美食と夜の冒険
夜

有名なドスハンバーガー

ボリューム満点のドスバーガー、アボカドバーガー、チリバーガーなどがおすすめ。高田パークで遊ぶ前に腹ごしらえに最適です。

✓ ポイント: ボリューム満点で夜のアクティビティに備えるのに最適なエネルギー補給スポット！

夜

高田パーク(遊園地)

ライトアップされたアトラクションが幻想的な雰囲気。絶叫系からファミリー向けまで楽しめます。夜ならではのイベントにも注目。

✓ ポイント: 夜空の下で楽しむアトラクションは昼間とは全く違った魅力があります！

2日目 - グルメとショッピング
朝

デュシットビーチリゾートホテルの朝食ビュッフェ

グアムのローカルフードを含む国際色豊かなメニュー。パン、卵料理、フルーツ、シリアル、ヨーグルトなどが楽しめます。

✓ ポイント: 1日の始まりにエネルギー満タン！海を眺めながらの朝食は格別です。

昼

アウトレット＆お土産探し

デュシットビーチリゾートホテルからアクセスしやすいGPO、マイクロネシアモール、タモンサンズプラザがお勧め。GPOは高級ブランド、マイクロネシアモールは幅広いジャンル、タモンサンズプラザは手頃な価格帯のショップが多いです。

✓ ポイント: ショッピング天国グアムでお気に入りのブランドをお得にゲット！

昼

タシグリル(めちゃ雰囲気⭕️レストラン)

新鮮なシーフードやステーキがおすすめ。ロブスターやフィレミニョンは特に人気です。

✓ ポイント: 雰囲気抜群のレストランで上質な食事を堪能しましょう！

夕方〜夜

ビーチでゆっくり & 好きなこと

夕日を背景にしたり、波打ち際で遊ぶ様子を撮影したり、ヤシの木と一緒の写真を撮るのがおすすめ。ホテル前のビーチ以外にも、恋人岬やイパオビーチも絶景スポットです。恋人岬は夕日の名所としても有名。イパオビーチは白い砂浜と青い海のコントラストが美しいです。

✓ ポイント: SNS映え間違いなし！グアムの美しい景色を背景に素敵な思い出を残しましょう。

夜

全社会食

旅行の思い出話に花を咲かせましょう。

✓ ポイント: 仲間との楽しい食事で2日目の締めくくり。旅の思い出を共有する素敵な時間です。

3日目 - 最後の冒険
朝

全員でパラセーリング

空から美しい海を眺める爽快な体験。撒布やビーチバレーなどもできる予定！！

✓ ポイント: 最高の景色を眺めながらのパラセーリングは一生の思い出に！

✓ ポイント: 最後の食事は特別な場所で。グアムの味を心に刻みましょう！
